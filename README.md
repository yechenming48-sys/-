# 玄机阁 · 东方神秘命理门户

静态网页合集，包含星轨神谕、紫微天枢、赛博周易三个命理系统。

## 本地预览

```bash
python3 -m http.server 8080
```

浏览器访问 `http://localhost:8080`

## GitHub Pages 公开访问

1. 合并包含站点文件的 PR 到 `main`
2. 打开仓库 **Settings → Pages**
3. **Source** 选择 `Deploy from a branch`
4. **Branch** 选择 `main`，目录选择 `/ (root)`
5. 保存后等待 1–3 分钟
